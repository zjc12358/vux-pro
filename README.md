# vux-pro

> 使用 vue + vue-router + webpack + vuex + mint-ui 实现

## 启动项目

```sh
安装依赖
$ npm install
启动项目
$ npm run dev
在浏览器中输入 http://localhost:8080
```
## 链接
* [demo](https://main.hzleshare.net/mobile/index.html#/)

## 项目目录
```sh
components // 自定义组件
assets // 静态文件资源
pages // 项目的页面view
router // 项目的路由定义
store // vuex的定义
utils // 项目自定义工具集合
```
